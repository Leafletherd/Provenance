# PR-18 — Provenance home screen parity with Seed

**Design rule:** Seed and Provenance share the same home screen layout. Every visual change made to Seed's home screen (SD-14 §4) must be mirrored here, substituting Provenance-specific tokens. The two screens should be indistinguishable in structure and feel.

This is a token-and-typography pass only. The layout and content are not changing.

## (2) Home screen typography and color tokens

**All token values must come from `BrandTokens.swift`, not hardcoded hex. Cross-reference `procestra_ui_color_assignment.html` for the exact token names.**

### 2a — Greeting typography

The Provenance greeting string (check-in prompt, home greeting, or equivalent top-of-screen text) must use Palatino Italic light — the in-app editorial serif equivalent to Fraunces (which is web-only and must not be bundled):

```swift
.font(.custom("Palatino-Italic", size: 34)).fontWeight(.light)
```

Fallback: `.font(.system(size: 34, weight: .light, design: .serif))`

This is the one string on the home screen that should use a serif. Everything else stays in system UI.

### 2b — Stat numbers

Large numbers on the Provenance home screen (word counts, commit counts, project stats, or whatever Provenance surfaces as prominent numbers) should use Palatino roman:

```swift
.font(.custom("Palatino", size: 28)).fontWeight(.regular)
```

### 2c — Primary action button

The primary action button on the home screen (equivalent of Seed's "Plant" button — probably the check-in or write action) must use:
- Background: `prov/accent` (`#1D9E75` light mode)
- Text: `#FAF6EF` (cream)
- Corner radius: match suite standard

The current treatment must be low-contrast. This should read as a clear, confident primary action.

### 2d — Eyebrow label

The small label above the projects or content strip must use `Brand.textMuted` (`#8C8A84` light mode) explicitly. Confirm it is not inheriting a system grey.

### 2e — Status/project dots or indicators

Any colored status indicators on the Provenance home screen:
- Active/healthy project indicator: `prov/accent` (`#1D9E75`)
- Inactive or secondary indicator: `Brand.textMuted` (`#8C8A84`)
- Muted/borderline indicator: `prov/tint-border` (check `procestra_ui_color_assignment.html` for the exact token — equivalent of Seed's `seed/tint-border`)

Replace any system colors with these explicit tokens.

### 2f — Sidebar selected row

The sidebar selected row background must use `prov/tint-surface` instead of the system grey highlight. Add a 0.5pt left accent stroke in `prov/tint-border`. Applies everywhere the sidebar appears.

```swift
.background(isSelected ? Color("prov/tint-surface") : Color.clear)
.overlay(alignment: .leading) {
    if isSelected {
        Rectangle()
            .fill(Color("prov/tint-border"))
            .frame(width: 0.5)
    }
}
```
